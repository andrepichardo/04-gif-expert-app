# GifExpertApp

A simple and friendly GIF Search Engine powered by GIPHY.

This project was developed and tested using React, TailwindCSS, Vite, Jest & React Testing Library.

# How to run

Clone repo
Run npm install
Run npm start

Check the project in action here:
